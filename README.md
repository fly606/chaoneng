# 《你的气味人格，藏着一座植物庄园》H5 互动页

副标题：**一场关于“香气疗愈”的沉浸式天然植愈之旅**。

本项目为「超能植愈香氛洗衣液」重新制作的原生 H5 互动页面。核心不是传统卖点陈列，而是把产品体验包装成「气味人格测试 + 沉浸式疗愈旅程 + 可分享社交结果」，让用户感知“这个香味，像在治愈我的生活”。

## 项目结构

```text
.
├── index.html        # H5 页面入口：首屏、测试、分析过渡、结果香卡
├── css/
│   └── styles.css    # 莫奈植物庄园视觉、动效、海报卡与移动端适配
├── js/
│   └── main.js       # 气味人格测试、结果计算、分享文案与页面切换
└── assets/           # 可放置品牌图、产品图、分享海报等素材
```

## 互动流程

1. **进入莫奈植物庄园**：首屏呈现主题名称与“香气疗愈”概念。
2. **气味人格测试**：用户依次选择最近情绪、穿衣习惯、理想生活状态、喜欢的气味场景。
3. **植物庄园唤醒**：以香氛瓶和加载动画承接测试到结果的情绪转场。
4. **生成植愈人格香卡**：输出植物香氛人格、疗愈文案、专属香气动画、超能产品推荐和分享文案。
