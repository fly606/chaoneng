# 超能植愈香氛系列 H5：天然香气疗愈之旅

本项目是一套原生 HTML + CSS + JavaScript 的移动端 H5 互动页，围绕「超能植愈香氛系列」构建从情绪识别、气味选择、植物调香到人格海报分享的完整体验链路。

核心创意不再是传统洗衣液卖点介绍，而是把产品转译成一次「洗情绪」的天然植愈旅程：用户的情绪会进入植物世界，气味偏好会被具象化，主动调香行为会生成专属植愈人格，最终沉淀为可分享的疗愈海报。

## 项目结构

```text
.
├── index.html        # 七页 H5 结构：首屏、情绪、气味、调香、人格、科技、分享
├── css/
│   └── styles.css    # 朦胧森林光影、莫奈花园、玻璃拟态、调香与海报视觉
├── js/
│   └── main.js       # 页面流转、情绪生态反馈、气味选择、拖拽调香、结果生成
└── assets/           # 可放置产品图、品牌图、音乐、分享图等素材
```

## 七页体验流程

1. **朦胧森林光影首页**：以“你有多久，没认真闻过生活的味道了？”建立情绪入口，配合花瓣、空气与微雾氛围。
2. **情绪进入植物世界**：用户选择焦虑之花、内耗之藤、压力之叶、孤独之花或治愈之芽，页面色彩、植物生态与光影同步反馈。
3. **气味选择环节**：围绕“你最想留在衣物上的味道”，提供雨后花园、阳光白衬衫、深夜木质香、清晨花园、被拥抱后的温度等气味场景。
4. **植物调香体验**：通过点击或拖拽白鸢尾、白兰、晚香玉、木质香调进行组合，模拟专属植愈香氛生成。
5. **植愈人格生成**：根据情绪、气味与调香数据，生成专属植愈香氛人格、植物香型、疗愈标签与产品推荐。
6. **产品技术力展示**：用香气分子、微胶囊释放与衣物纤维场景，呈现 5 重香氛专利技术、720 小时持久留香体系、双重异味中和机制与植物净护成分。
7. **最终分享海报**：生成轻奢香氛海报风格的专属“植愈香氛疗愈海报”，支持复制分享文案。

## 本地预览

直接用浏览器打开 `index.html`，或启动静态服务：

```bash
python3 -m http.server 4173
```

访问：<http://localhost:4173>

## 交互说明

- 情绪卡片会影响全局背景色、植物生态光影和后续人格结果。
- 气味选择会影响香气视觉色彩与结果海报文案。
- 调香页面支持桌面端拖拽，也支持移动端点击添加香调。
- 至少选择 2 种植物香调后，才可生成植愈人格。
- 分享页会将用户的人格名称、情绪植物与专属香调整合成可复制文案。
