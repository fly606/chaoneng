# 超能植愈香氛洗衣液 H5 互动页

一个轻量级 H5 互动页面项目，使用原生 HTML、CSS 与 JavaScript 构建，主题为「超能植愈香氛洗衣液」。页面包含首屏产品展示、植萃能量收集、香氛选择与洗护报告生成等互动环节。

## 项目结构

```text
.
├── index.html        # H5 页面入口
├── css/
│   └── styles.css    # 页面视觉、动效与移动端适配
├── js/
│   └── main.js       # 页面切换、触摸滑动与互动逻辑
└── assets/           # 可放置品牌图、产品图、分享海报等素材
```

## 本地预览

直接用浏览器打开 `index.html`，或启动一个静态文件服务：

```bash
python3 -m http.server 4173
```

然后访问：<http://localhost:4173>

## 互动亮点

- **沉浸式首屏**：通过 CSS 绘制洗衣液瓶身、泡泡、光晕与植物氛围。
- **植萃能量收集**：点击三张植萃卡片，逐步点亮洁净、柔护、留香能力。
- **香氛选择器**：滑动选择晨露、白茶、铃兰、木质等香气场景。
- **结果报告**：根据香氛选择生成洗护香氛指数与推荐文案。
- **移动端体验**：支持点击圆点导航、键盘方向键与移动端上下滑动切换页面。
